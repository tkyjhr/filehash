# filehash.go

`filehash.go` defines functions that calculate the hash value of a given file.

```go
sha1 := filehash.SHA1("somefile.txt")
fmt.Printf("%x", sha1)
```

# cmd/filehash/main.go

`go install` under `cmd/filehash` generate a command line tool.


## Example usage

```

8b3359b31039237eb30ffe76a430ee48ebbf476b

```
